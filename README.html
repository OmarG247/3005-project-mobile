Overview

This project is a mobile app built with React Native with an SQLite implementation built for the expo sdk.

Design process

This application was first designed and prototyped using Figma.

You may find try out the prototype here.

Technologies used

  • React Native
  • SQLite
  • React Redux

Try it out!

To run this on your locally phone, you'll have to do the following:

  • Install the expo app on your mobile device through either the Google Play Store or the Apple Store
  • Clone this repository
  • Install node & expo-cli
    • download node here
    • install the expo cli $ npm i -g expo-cli
  • Run $ npm start
  • Follow the instructions shown on the expo tab on your browser

Alternatively, you may download the apk file for android here.
